# Internship-portal-java
## Overview
This is a Spring Boot application for managing internship programs, including user management, internship tracking, task management, and feedback collection.

## Project Structure
- **Controller Layer**: Handles HTTP requests (User, Internship, Task, Feedback)
- **Service Layer**: Business logic implementation
- **Repository Layer**: Database access using Spring Data JPA
- **Model Layer**: Entity classes for domain objects

## Getting Started
Run the application using Maven:
```bash
mvn spring-boot:run
```
